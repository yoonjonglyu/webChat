# webChat
> 리액트 채팅 컴포넌트 모듈 만들기 + 해당 모듈 데모 버전

## 목적
> 채팅 모듈을 npm에 배포하기전에 미리 만들어서 기능 구현이나 테스트 등을 하는게 목적이다. 
> 추가로 해당 모듈 패키지 배포 이후 해당 레포는 해당 모듈 패키지을 체험할 수 있는 데모 버전으로 사용한다.
